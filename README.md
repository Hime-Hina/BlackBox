## 查看Demo

  1.克隆仓库

  输入命令`git clone https://github.com/Hime-Hina/BlackBox.git`.

  2.安装依赖

  首先得安装`typescript`, 运行`npm install -g typescript`.

  然后, 在`BlackBox`目录下运行`npm install`.

  3.运行

  运行`webpack`, 将会创建`dist`目录, 用浏览器打开`index.html`即可.
